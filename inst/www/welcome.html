simplex is a free and open data reduction program for Secondary Ion Mass Spectrometery (SIMS) that offers the following features:
  1. simplex processes mass spectrometer in logratio space (a.k.a. 'the simplex'), ensuring strictly positive isotopic ratios with appropriate (asymmetric) uncertainty estimates.
  2. Secondary Electron Multiplier (SEM) data are modelled with Poissonian statistics, ensuring adequate handling of zero count data.
  3. Error propagation is done in matrix form, and accounts for all source of systematic uncertainty, including inter-sample error correlations.

simplex handles data from both Cameca and SHRIMP instruments, using both single collector and (for Cameca) multicollector configurations, for either U-Pb, Th-Pb or U-Th-Pb geochronology, or stable isotope geochemistry. The program can be used in three ways:
  1. Online: This website allows installation-free access to the Graphical User Interface (GUI). It includes a number of built-in datasets that allow novice users to familiarise themselves with the software. The online GUI is less well suited for routine processing of large datasets.
  2. Offline: For a more responsive user experience, simplex can also be installed on any computer (running Windows, Mac-OS or Linux), within the R statistical programming environment. To this end, install R on your system (see https://r-project.org/ for details), and then enter the following commands at the R console: 
    	install.packages('remotes')
	remotes::install_github('pvermees/simplex')
    Once installed, the offline GUI can be started as follows: 
    	simplex::simplex()
  3. Command line: Advanced users can write automation scripts or extend the functionality of the simplex package by calling it from the command line, using the R programming language. A detailed Application Profile Interface (API) is accessible from the console. To get started, type: 
    	help(package="simplex")
