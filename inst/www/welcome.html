simplex is a free and open data reduction program for Secondary Ion Mass Spectrometery (SIMS) that processes the measurements in logratio space (a.k.a. "the simplex").

The advantages of this approach are described in the following manuscript:

Vermeesch, P. 2022, An algorithm for U-Pb geochronology by secondary ion mass spectrometry. Geochronology, 4, 561-576, doi:10.5194/gchron-4-561-2022.
simplex handles data from both Cameca and SHRIMP instruments, using both single collector and (for Cameca) multicollector configurations, for either U-Pb, Th-Pb or U-Th-Pb geochronology, or stable isotope geochemistry. The program can be used in three ways:
  1. Online: This version allows installation-free access to the Graphical User Interface (GUI). It includes a number of built-in datasets that allow novice users to familiarise themselves with the software. The online GUI is less well suited for routine processing of large datasets.
  2. Offline: For a more responsive user experience, simplex can also be installed on any computer (running Windows, Mac-OS or Linux), within the R statistical programming environment. To this end, install R on your system (see https://r-project.org/ for details), and then enter the following commands at the R console: 
     install.packages('remotes')
 remotes::install_github('pvermees/simplex')
    Once installed, the offline GUI can be started as follows: 
     simplex::simplex()
  3. Command line: Advanced users can write automation scripts or extend the functionality of the simplex package by calling it from the command line, using the R programming language. A detailed Application Profile Interface (API) is accessible from the console. To get started, type: 
     help(package='simplex')
All three methods follow the same sequence of steps, corresponding to the six tabs at the top of this page.

simplex is beta software that has not yet undergone rigorous testing. Bugs and suggestions can be reported on GitHub, or by emailing p.vermeesch[at]ucl.ac.uk. Thanks for the feedback!

Example datasets were provided by Dr. Yang Li (IGG-CAS Beijing) and Dr. Simon Bodorkos (Geoscience Australia). The development of simplex was supported by NERC standard grant #NE/T001518/1 ('Beyond Isoplot').
