Four-step calibration of SIMS data.
I. Calibration type - Choose one of two options:
1. isotopic fractionation:
The default setting for multi-collector stable isotope data. Accounts for the mass bias of isochemical isotope ratios such as 18O/16O. Uses the logratio mean of a reference standard to correct the sample.
2. elemental fractionation:
The default setting for single isotope geochronology data. Accounts for the chemical fractionation of multi-element isotopic ratios such as 206Pb/238U. Uses a calibration curve of the form Pb/U vs. UO/U.
It is usually not necessary to change the default option.
II. Standard composition - Choose one of three options:
1. specify manually:
Manually enter the logratio composition of the reference material, as well as its covariance matrix and a table of text labels that pair the standard to the measurements. The latter table contains the following headers:
X: The logratio to be used as the independent variable (x-axis) of the calibration curve.
Y: The logratio to be used as the dependent variable (y-axis) of the calibration curve.
C: The logratio to be used as a proxy for common Pb. Only relevant if the common+radiogenic option is selected (see below for details).
slope: Either 'auto' or a numerical value, which fixes the slope of the Pb/U-UO/U calibration curve.
2. load presets:
Selecting this item of the pull-down menu adds a new pull-down menu to the user interface, containing a selection of commonly used reference materials with preloaded values. Selecting one of these options populates the three tables with those values.
3. convert from age:
Selecting this item of the pull-down menu adds two text boxes to the user interface, specifying the age of the reference material and its standard error. These values can be converted to a set of equivalent logratio compositions.
For each of these options, the standard composition can either by specified as:
1. common+radiogenic:
The U-Pb composition of discordant reference materials can be specified as a mixing line between purely radiogenic and purely non-radiogenic end members.
2. measured:
Alternatively, the composition of reference materials can also be specified as the measured U-Pb composition of the standard. This requires that the isotopic composition of the standard tightly clusters around a specific set of values. Any spread of the reference data along a discordia line cannot be adequately captured by a single composition.

3. convert from δ-values:
Selecting this item of the pull-down menu adds three new items to the user interface:
δ-values: A table with the isotopic composition of the reference material, specified inconventional stable isotope notation. (δ = (Rsmp/Rstd - 1) x 1000 where Rsmp and Rstd are isotopic ratios measured in the sample and reference standard, respectively)
Covariance matrix: The covariance matrix of the δ-values.
Reference material: The isotopic ratios of the reference standard (i.e. Rstd above). A pull-down menu of commonly used reference standards is provided to populate this table.
III. Select standard aliquots:
Enter the prefix of the standard in the text box to select the aliquots to be used in the calibration curve. Alternatively (or additionally), individual aliquots can also be added or removed by entering 'yes' or 'no' in the appropriate cells of the table.
IV. Calibrate:
Click the button to construct a calibration curve, which is needed for subsequent steps in the data reduction chain. The row numbers of the table of aliquots can be added as labels to the calibration curve. This facilitates the identification of outliers, which can be removed by revisiting step III of the calibration procedure.
