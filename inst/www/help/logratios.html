Fit a log-linear model to the drift-corrected logratio data, using a Poisson error model for SEM data.
Aliquot:
Select an aliquot from the pull-down menu or cycle through them by pressing the < and > buttons.
Plot:
Plot the time-resolved blank corrected signals logratios for the selected aliquot. Produces a multi-panel plot of time resolved signals or ratios, with a black line that shows the best fit log-linear fit to the data (where log[x/y] = b0 + g x t).
auto:
If ticked, automatically refreshes the logratio plot when moving up ('>') or down ('<') the sample list.
ratios:
If unticked, Plot produces one plot for each ion in the Numerators and Denominators fields of the Setup page. If ticked, Plot produces one plot for each ratio in the Numerators and Denominators fields of the Setup page.
Tabulate:
Tabulates the estimated (log)ratios for the entire dataset. Produces a table of estimates and uncertainties that includes all the within-spot error correlations. Aliquots whose backgrounds exceed some signals (typically 204Pb) are marked with an asterisk (*).
log:
If ticked, the output table reports the compositions as logratios.
+x/y:
If ticked, adds the x-y stage coordinates of the aliquots to the table. Only available for Cameca data.
Omit:
A comma-separated list of 'outliers' to omit from the datasets, corresponding to the row numbers of the Signal and Time tables below.
b0:
The intercepts of the log-linear ratio drift model
g:
The slopes of thelog-linear ratio drift model. g=0 for isochemical ratios.
