Prepare the calibrated results for postprocessing in other software.
Convert to IsoplotR:
Recasts the data in a format that can be read by IsoplotR. Depending on the isotopes analysed, the following formats are possible:
U-Pb: Corresponds to IsoplotR U-Pb format 5 (X=38/06, err[38/06], Y=07/06, err[Y], Z=04/06, err[Z], rho[X,Y], rho[X,Z], rho[Y,Z]).
Th-Pb: Corresponds to IsoplotR Th-Pb format 2 (X=32/08, err[X], Y=04/08, err[Y], rho).
U-Th-Pb: Corresponds to IsoplotR U-Pb format 8 (W=38/06, err[W], X=07/06, err[X], Y=08/06, err[Y], Z=32/38, err[Z], rhoWX, rhoWY, rhoWZ, rhoXY, rhoXZ, rhoYZ).
Export:
Saves the output in a .json format that can be opened in the online version of IsoplotR.
Convert:
Convert the logratio data into δ-values, using one of the following two conventions:
δ: The conventional definition, i.e. δ = (Rsmp/Rstd - 1) ‰, where Rsmp and Rstd are isotopic ratios measured in the sample and reference standard, respectively.
δ': The logratio definition of Hulston and Thode (1965, JGR, 70.14, 3475-3484), i.e. δ' = (ln[Rsmp]-ln[Rsmp])‰.
Reference material:
The isotopic ratios of the reference standard (Rstd) that is used for δ-notation (see above). A pull-down menu of commonly used reference standards is provided to populate the table with default values.
